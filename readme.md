# Моя новостная страница

Разработан в качестве практического задания на курсе Веб разрабодчика от [Skillfactory](https://skillfactory.ru)

## Используемые технологии
* HTML
* CSS
* Web-fonts (Manrope, TildaSans, Arial)
* GitHub

## Задачи и цели
Проект был направлен на создание простой новостной страницы с использованием фундаментальных инструментов веб‑разработки — HTML и CSS, в том числе селекторов с псевдоклассами и псевдоэлементами.

### Техническое задание
- Написать HTML и CSS код из примера
- Дополнить количество новостей, чтобы их было 5
- Заменить и дополнить контент
- Использовать внешние ресурсы для блоков с фотографиями
- Используя селектор псевдоэлемента добавить к заголовкам пeрвых 2х новостей слово "New".
- Используя селектор псевдокласса задать цвет фона у каждой второй (четной) новости (цвет на ваше усмотрение).
- Оформить файл README с коротким описанием проекта
- Разместить проект в GitHub и создать Git Pages

## Вывод
Это интересный проект, который помог мне разобраться с псевдоклассами, приоритетами селекторов и многое другое.

Из-за этого у меня постоянно нарушалась иерархия и стиль иногда переставал работать. Помимо этого, при публикации страницы в GitHub и Git Pages у меня перестали грузиться фотографии на странице, пока не включишь VPN. Я так понимаю, что сервера сайта [unsplash](https://unsplash.com) перестали работать на территории РФ.

Своей работой я доволен и хочу скорее набраться опыта и набить руку, чтобы самостоятельно верстать проекты, которые до этого я делал в конструкторе Тильда.
